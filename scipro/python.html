2.2. Python and Anaconda¶

2.2.1. Python¶

Python is a cross-platform object-oriented programming language which gained its popularity in the recent years.

It is a high-level programming language, which means:

  • It has a simple and user friendly syntax.

It has a huge community, including the astronomical community. That is to say:

  • There are many useful libraries/scripts available, many following the PEP8 (style guideline for python).

  • It is relatively easy to find answers to the question/issue/problem you run into on Google or Stack Overflow.

2.2.2. Anaconda¶

Anaconda is a free and open source distribution of Python. It has a set of most popular packages in the scientific community. It also has an environment manager (conda).
